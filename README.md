# SALES PERFORMANCE DASHBOARD

## Project Overview:
The objectives of building a dashboard was to build visuals that explain:
1. Sales performance.
2. Product performance.
3. Customer insights.
4. Inventory insights.

## ools Used
1. Power BI
2. Data Cleaning Techniques
3. Data Modeling
4. DAX Calculations

## KPIs included were:
1. Total Sales Revenue
2. Total Orders
3. Monthly Sales Trends
4. Profit Analysis
5. Sales Growth Rate

## Visuals used: 
1. Line Charts
2. Bar Charts
3. KPI Cards
4. Area Charts

## Generated Insights:
1. Best-performing sales periods
2. Revenue growth trends
3. Seasonal sales patterns
4. High and low sales months
